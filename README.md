## О проекте

Классический калькулятор, выполненный в современном стиле. Подходит для выполнения повседневных математических вычислений. Интерфейс интуитивно понятен и адаптируется под размер экрана устройства.

## Функциональность

- **Базовые арифметические операции** — сложение, вычитание, умножение, деление
- **Смена знака числа** — преобразование положительного числа в отрицательное и наоборот
- **Вычисление процентов** — автоматический расчет процента от введенного числа
- **Очистка экрана** — кнопка AC для сброса всех введенных данных
- **Адаптивная верстка** — корректное отображение на ПК, планшетах и мобильных устройствах
- **Эффекты при нажатии** — визуальная обратная связь при взаимодействии с кнопками

## Используемые технологии

- **HTML5** — семантическая структура
- **CSS3** — Flexbox, Grid, медиазапросы, hover-эффекты
- **JavaScript** — обработка событий, вычисление выражений через eval()
